Using PlantUML in Hugo

PlantUML is a fantastic tool for creating diagrams using a simple text-based description language. This post demonstrates how to use PlantUML in your blog posts.

Use Cases

PlantUML is great for:

  1. Software architecture documentation
  2. Database schema design
  3. Sequence flows
  4. Class relationships
  5. State diagrams

How It Works

When you write a blog post, you can include PlantUML diagrams like this:

{{< plantuml >}}
@startuml
Alice -> Bob: Hello
@enduml
{{< /plantuml >}}

The Hugo shortcode converts your PlantUML code into an encoded URL for the PlantUML server, which returns the rendered diagram as an image.

Advanced Example: Activity Diagram

This approach leverages the online PlantUML server to render your diagrams. For high-volume or private sites, you might want to consider setting up your own PlantUML server.
